删除节点 

单分支删除d节点：
  a                a
 / \              / \
T0  b            T0  b
   / \     =>       / \
  T1  d            T1  c (d)
     /
    c
双分支删除d节点：
   d                  c
 /   \              /   \
T0    b            T0    b
    /   \     =>       /   \
   c     T1           d     T1
    \   / \            \   / \
     a T2 T3            a T2 T3
节点c为节点d直接后继，交换c和d，然后按单分支情况删除d

node待删除的目标节点

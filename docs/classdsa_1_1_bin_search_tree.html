详细描述

template<typename T>
class dsa::BinSearchTree< T >

二叉搜索树类模板 

节点数: n
树高度: h，最坏情况下为n
search, insert, remove在最坏情况下需要O(h)时间。

树结点联接 

最终的树如下所示：
      b
    /   \
   a     c
  / \   / \
 T0 T1 T2 T3
同时更新a,b,c三个节点的高度（connect34会调用update_height()函数）；
而T0,T1,T2,T3，因为其子节点未发生变动，所以其高度也就无需更新；

节点旋转调整 


示意图如下(包括对称情况)：
单旋:
zag(旋转gp)                       | zig(旋转gp)
  g                       p       |        g                       p
/   \                   /   \     |      /   \                   /   \
T0   p       =>        g     v    |     p    T3     =>          v     g
    / \               / \   / \   |    / \                     / \   / \
   T1  v             T0 T1 T2 T3  |   v  T2                   T0 T1 T2 T3
      / \                         |  / \
     T2 T3                        | T0 T1
                                  |
双旋:
zig-zag(先zig旋转pv，再zag旋转gv) | zag-zig(先zag旋转pv，再zig旋转gv)
  g                       v       |      g                       v
/   \                   /   \     |    /   \                   /   \
T0   p                 g     p    |   p    T3                 p     g
    / \               / \   / \   |  / \                     / \   / \
   v  T3     =>      T0 T1 T2 T3  | T0  v           =>      T0 T1 T2 T3
  / \                             |    / \
 T1 T2                            |   T1 T2
                                  |
